## 📌 Introduce

> **Kimchi** : 한국을 방문한 외국인 관광객들을 위한 여행 사진 소셜 플랫폼

하나, 둘, 셋 ... 김치   
Kimchi는 한국에서 시작된 독특한 사진 촬영 스타일인 네컷 사진을 통해   
관광객들이 자신만의 개성과 매력을 사진으로 표현할 수 있는 플랫폼입니다.
